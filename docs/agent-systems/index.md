# Agent Systems

Agent systems are useful when a workflow needs decomposition, tool choice, state, and controlled autonomy. The hard part is not making an agent act. It is making the system observable, governable, and safe to operate.

## In this section

- [Agent Foundations](../foundations/agent-foundations.md) defines when agentic design is justified.
- [Internal Coding Agent](internal-coding-agent.md) covers the shared architecture behind internal coding copilots and higher-autonomy coding agents.
- [Multi-Agent Orchestration](multi-agent-orchestration.md) explains when multiple agents help and when they create needless complexity.
