# Model Lifecycle

This section covers the systems that manage model change over time. The focus is not model research. It is the operational pipeline for datasets, experiments, evaluation, promotion, and rollback when a team fine-tunes or otherwise changes a model-backed capability.

## In this section

- [Design an LLM Fine-tuning System](llm-fine-tuning-system.md) covers data preparation, training orchestration, evaluation gates, and release control.

## Section lens

If product chapters ask what a system does, model lifecycle chapters ask how changes to the underlying model are introduced safely.
