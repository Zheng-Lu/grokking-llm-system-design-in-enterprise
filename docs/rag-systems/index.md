# RAG Systems

RAG systems sit at the intersection of document operations, search quality, permissions, and grounded generation. In enterprise settings, that combination is where many seemingly simple assistants become difficult to design well.

## In this section

- [Foundations](foundations.md) explains the core architecture and the main enterprise failure modes.
- [Permission-Aware RAG](permission-aware-rag.md) covers retrieval with authorization boundaries.
- [Hybrid Retrieval](hybrid-retrieval.md) shows when lexical and semantic search should work together.
- [Design Prompt: RAG System](design-prompt.md) compresses the topic into an interview-style system design exercise.
