# Vertical Applications

Vertical applications combine the lower-level patterns from the rest of the book into a domain-specific product surface. This section is where retrieval, platform infra, agents, and governance come together in one concrete system.

## In this section

- [Application Patterns](application-patterns.md) covers the product layer around enterprise LLM systems.
- [Design an AI-Based Shopping Chatbot](ai-shopping-chatbot.md) is a starter vertical application chapter that combines retrieval, recommendation logic, and transactional safety concerns.

## Section lens

A good vertical application chapter should make it obvious which lower-level systems it depends on and where the user experience changes the architecture.
